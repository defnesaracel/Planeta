# 🌍Planeta
## 📖About Our Project
**Planeta** is a web application that aims to help users realize the environmental impact of their daily actions and encourages them to have a more sustainable lifestyle. Users are evaluated with a scoring system based on their answers to daily surveys. Users will be able to get feedback based on their answers and get weekly/monthly reviews to track their progress. They will be awarded with badges based on their scores.
